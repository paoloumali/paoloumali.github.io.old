## Alias commands

- do: 
  ```bash
  chmod +x ./d
  alias d="./d"
  ```
- $ ``d up``, to serve
- $ ``d build``, to build prod
- $ ``d``, to check stat

## Local setup

- $ ``d install``
- $ ``d serve``
- Browse http://localhost:4000
